【第19章】账面价值倍数：从 PBV 到 Tobin’s Q 的价值发现

投资
估值
深入解析市净率（PBV）、企业价值/账面资本比率和托宾Q的内在逻辑，揭示ROE如何成为账面价值倍数的核心驱动因素，以及如何在实践中发现价值投资机会。
作者

Ying

发布于

2025年12月19日

1 从一个问题开始

假设你在筛选股票时发现一家公司，股价只有账面价值的 0.8 倍——也就是说，你可以用 80 分钱买到 1 块钱的”净资产”。这是捡便宜的绝佳机会吗？

很多投资者会本能地认为”低于账面价值就是低估”，但这种直觉往往是危险的。一家公司的股价低于账面价值，可能恰恰是因为它的资产根本不值账面价值那么多——如果公司的资产回报率（ROE）低于股东要求的回报率，那么每投入一块钱的账面资本，实际上在损毁价值。

市净率（Price-to-Book Value Ratio，PBV）是价值投资者最钟爱的指标之一，但也是最容易被误用的指标之一。 从本杰明·格雷厄姆到沃伦·巴菲特，价值投资大师们都强调”用低于账面价值的价格买入”，但他们同时也强调”护城河”——也就是公司维持高 ROE 的能力。

本章将深入探讨：

  • PBV 的真正决定因素是什么？为什么 ROE 是核心？
  • 如何从 DCF 模型推导 PBV 与基本面的关系？
  • 何时低 PBV 是低估，何时只是对低质量的合理定价？
  • 企业价值/账面资本比率与 PBV 有什么区别？何时更适用？
  • 托宾 Q（Tobin’s Q）这个学术概念有什么实际意义？

2 市净率（PBV）：账面价值的市场定价

2.1 什么是账面价值？

在深入 PBV 之前，我们需要理解”账面价值”（Book Value）这个概念。

股权的账面价值 = 资产的账面价值 − 负债的账面价值

在美国会计准则下，资产的账面价值是原始购买价格减去累计折旧。这意味着：

  1. 账面价值反映的是历史成本，而非当前市场价值
  2. 随着资产老化，账面价值会下降
  3. 如果资产的盈利能力显著变化（上升或下降），账面价值可能与真实价值严重脱节
注记账面价值的”历史成本”本质

想象一家公司 10 年前以 100 万美元购买了一台设备，按 10 年直线折旧。今天的账面价值是 0，但这台设备可能仍在高效运转，每年创造 20 万美元的现金流。

反过来，另一家公司 5 年前以 500 万美元购买了最新技术，账面价值还有 250 万，但这项技术已经过时，几乎没有产出。

这就是账面价值与真实价值脱节的典型例子。

2.2 PBV 的定义与变体

市净率的基本定义非常简单：

\[ \text{PBV} = \frac{\text{每股市价（Price per Share）}}{\text{每股账面价值（Book Value of Equity per Share）}} \]

或者用总量形式：

\[ \text{PBV} = \frac{\text{股权市值（Market Capitalization）}}{\text{股权账面价值（Book Value of Equity）}} \]

重要PBV 的一致性要求

PBV 是一个股权倍数——分子是股权市值，分母是股权账面价值。计算时需要注意：

  • 不要包含优先股：账面价值应只包含普通股股东权益
  • 多类别股票：如果有多类股票，应使用所有普通股的总市值和总账面价值
  • 期权稀释：技术上应将管理层期权和可转债期权的市值加入分子

2.3 为什么分析师使用账面价值？

投资者使用 PBV 有几个原因：

优点 说明
稳定性 账面价值比盈利更稳定，不会像 EPS 那样剧烈波动
直观性 提供一个简单的”成本基准”来对比市场价格
普适性 即使盈利为负的公司也能计算 PBV（负账面价值的公司远少于负盈利的公司）
跨公司可比 在会计准则一致的前提下，可以跨公司比较

但 PBV 也有明显的缺陷：

缺陷 说明
会计差异 折旧方法、资产减值等会计选择影响账面价值
跨国不可比 不同国家的会计准则差异导致账面价值不可比
对服务/科技公司意义有限 缺乏有形资产的公司，账面价值意义不大
可能为负 持续亏损的公司账面价值可能为负

2.4 回购和收购对 PBV 的影响

近年来，股票回购和并购对账面价值产生了显著影响，值得特别关注。

股票回购的影响：

当公司回购股票时，账面权益会减少相应金额。举个例子：

  • 公司市值 1 亿美元，账面价值 5000 万美元，PBV = 2.0
  • 公司借款 2500 万美元回购股票
  • 市值和账面价值都下降 2500 万美元
  • 新 PBV = 7500 万 / 2500 万 = 3.0

回购”人为”提高了 PBV！这意味着在比较积极回购与不回购的公司时，需要谨慎。

收购的影响：

收购时，购买价格超过目标公司账面价值的部分会计入商誉（Goodwill）。商誉的处理方式会影响账面价值和 PBV：

  • 商誉需要定期进行减值测试
  • 减值损失会降低账面价值
  • 大量收购的公司账面价值中可能包含大量商誉
提示调整方法

如果要跨公司比较 PBV，可以考虑：

  1. 剔除商誉：使用有形账面价值（Tangible Book Value）
  2. 加回回购：将历史回购金额加回账面价值

\[ \text{调整后账面价值} = \text{账面价值} - \text{商誉} + \text{历史回购金额} \]

2.5 PBV 的跨截面分布

理解 PBV 在市场中的分布特征至关重要。图 19.1 展示了 2024 年 1 月美国和全球公司的 PBV 分布：

统计量 美国 全球
平均值 9.94 6.15
中位数 1.62 1.49
最大值 极高 极高
负账面价值公司数 ~2,000 更多

几个关键观察：

  1. 严重右偏分布：平均值远高于中位数，因为存在极高 PBV 的公司
  2. 负账面价值：美国约 2,000 家公司账面价值为负，无法计算 PBV
  3. 负账面价值的原因：多数情况下是股票回购和资产减值的结果，而非真正陷入困境

3 PBV 的决定因素：从 DCF 推导

3.1 稳定增长公司的 PBV

要理解 PBV 的真正驱动因素，我们需要回到 DCF 模型。从稳定增长的股息折现模型出发：

\[ \text{股权价值} = \frac{\text{预期股息}}{\text{股权成本} - \text{预期增长率}} = \frac{\text{净利润} \times \text{派息率}}{k_e - g} \]

定义 ROE = 净利润 / 期初账面价值，将净利润改写为：

\[ \text{净利润} = \text{账面价值} \times \text{ROE} \]

代入得：

\[ \text{股权价值} = \frac{\text{账面价值} \times \text{ROE} \times \text{派息率}}{k_e - g} \]

两边同除以账面价值，得到 PBV 的基本公式：

\[ \text{PBV} = \frac{\text{ROE} \times \text{派息率}}{k_e - g} \]

进一步，利用增长率与 ROE 的关系：

\[ g = (1 - \text{派息率}) \times \text{ROE} \]

可以推导出更简洁的形式：

\[ \boxed{\text{PBV} = \frac{\text{ROE} - g}{k_e - g}} \]

重要核心洞察：PBV = 1 的临界条件

从公式可以看出：

  • 当 ROE = \(k_e\)（股权成本）时，PBV = 1
  • 当 ROE > \(k_e\) 时，PBV > 1——公司创造超额回报，市价高于账面
  • 当 ROE < \(k_e\) 时，PBV < 1——公司损毁价值，市价低于账面

PBV 的本质是市场对公司创造超额回报能力的定价。

3.2 Vodafone 案例：稳定增长公司的 PBV 估算

让我们用 Vodafone 的数据来演示这个公式。假设 Vodafone 已进入稳定增长阶段：

参数 数值
净利润（2010） 7,968 百万英镑
股息（2010） 4,468 百万英镑
账面价值（2009年末） 90,810 百万英镑
派息率 4,468 / 7,968 = 55.82%
ROE 7,968 / 90,810 = 8.77%
预期增长率 (1 − 0.5582) × 8.77% = 3.88%
无风险利率 4%
Beta 1.0
股权成本 4% + 1.0 × 5% = 9.0%

计算 PBV（两种方法结果相同）：

\[ \text{PBV} = \frac{\text{ROE} \times \text{派息率}}{k_e - g} = \frac{0.0877 \times 0.5582}{0.09 - 0.0388} = 0.96 \]

\[ \text{PBV} = \frac{\text{ROE} - g}{k_e - g} = \frac{0.0877 - 0.0388}{0.09 - 0.0388} = 0.96 \]

解读：Vodafone 的 PBV 略低于 1，因为其 ROE（8.77%）低于股权成本（9%）。市场合理地给予其低于账面价值的定价。

3.3 Jenapharm 案例：私有化估值

1991 年德国统一后，Treuhandanstalt（德国私有化机构）需要为东德制药公司 Jenapharm 定价。这是 PBV 在私有化中的实际应用：

参数 数值
预期净利润（1991） 900 万德国马克
账面价值（1990年末） 5,800 万德国马克
ROE 9 / 58 = 15.52%
预期增长率 5%（长期）
无风险利率 7%
Beta 1.25（高于行业平均 1.05，反映更高风险）
股权成本 7% + 1.25 × 3.5% = 11.375%

计算 PBV：

\[ \text{PBV} = \frac{\text{ROE} - g}{k_e - g} = \frac{0.1552 - 0.05}{0.11375 - 0.05} = 1.65 \]

估计市值：

\[ \text{市值} = \text{PBV} \times \text{账面价值} = 1.65 \times 5800 = 9570 \text{ 万德国马克} \]

3.4 高增长公司的 PBV

对于高增长公司，我们需要使用两阶段模型。从两阶段股息折现模型推导：

\[ \frac{\text{Price}}{\text{Book}} = \text{ROE}_{hg} \times \left[ \frac{\text{派息率}_{hg} \times \left(1 - \frac{(1+g)^n}{(1+k_{e,hg})^n}\right)}{k_{e,hg} - g} + \frac{(1+g)^{n-1}(1+g_{st}) \times \text{派息率}_{st}}{(k_{e,st} - g_{st})(1+k_{e,hg})^n} \right] \]

其中：

  • \(\text{ROE}_{hg}\) = 高增长期的 ROE
  • \(g\) = 高增长期的增长率
  • \(g_{st}\) = 稳定期的增长率
  • \(k_{e,hg}\), \(k_{e,st}\) = 高增长期和稳定期的股权成本
  • \(n\) = 高增长期年数

PBV 的决定因素：

因素 对 PBV 的影响
ROE 正相关——ROE 越高，PBV 越高
派息率 正相关（给定增长率）
风险（通过 \(k_e\)） 负相关——风险越高，PBV 越低
增长率 正相关——增长越快，PBV 越高

3.5 高增长公司案例：两阶段模型

假设一家公司具有以下特征：

参数 高增长期（5年） 稳定期
EPS 增长率 20% 4%
派息率 20% 60%
ROE 25% 10%
Beta 1.0 1.0
股权成本 9.5% 9.5%

验证增长率的一致性： - 高增长期：\(g = (1 - 0.20) \times 25\% = 20\%\) ✓ - 稳定期：\(g = (1 - 0.60) \times 10\% = 4\%\) ✓

计算 PBV：

\[ \frac{\text{Price}}{\text{Book}} = 0.25 \times \left[ \frac{0.20 \times \left(1 - \frac{(1.20)^5}{(1.095)^5}\right)}{0.095 - 0.20} + \frac{(1.20)^4(1.04)(0.60)}{(0.095 - 0.04)(1.095)^5} \right] = 4.01 \]

这家高增长公司的 PBV 为 4.01，远高于稳定增长公司的水平。

3.6 Nestle 案例：用 FCFE 替代股息

如果公司不按照其支付能力支付股息，我们可以用 FCFE 替代股息。以 Nestle 为例：

参数 高增长期（5年） 稳定期
ROE 21.35% 10%
股权再投资率 37.17% 25%
FCFE/净利润 62.83% 75%
预期增长率 7.94% 2.5%
股权成本 6.90% 6.90%

计算 PBV：

\[ \frac{\text{Price}}{\text{Book}} = 0.2135 \times \left[ \frac{0.6283 \times \left(1 - \frac{(1.0794)^5}{(1.069)^5}\right)}{0.069 - 0.0794} + \frac{(1.0794)^4(1.025)(0.75)}{(0.069 - 0.025)(1.069)^5} \right] = 4.27 \]

4 PBV 与 ROE 的核心关系

4.1 ROE 变化对 PBV 的影响

ROE 的变化对 PBV 有双重影响：

  1. 直接影响：通过 PBV 公式中的 ROE 项
  2. 间接影响：通过改变可持续增长率 \(g = (1 - \text{派息率}) \times \text{ROE}\)

让我们看一个具体例子。假设前面那家高增长公司的 ROE 从 25% 下降到 12%：

新的增长率：\(g = 0.8 \times 12\% = 9.6\%\)（而非 20%）

新的 PBV：

\[ \frac{\text{Price}}{\text{Book}} = 0.12 \times \left[ \frac{0.20 \times \left(1 - \frac{(1.096)^5}{(1.095)^5}\right)}{0.095 - 0.096} + \frac{(1.096)^4(1.04)(0.60)}{(0.095 - 0.04)(1.095)^5} \right] = 1.36 \]

PBV 从 4.01 暴跌至 1.36！ROE 下降 13 个百分点（从 25% 到 12%），导致 PBV 下降约 66%。

4.2 PBV 与超额回报的关系

图 19.2 展示了 PBV 与”超额股权回报”（ROE − 股权成本）之间的关系：

ROE − 股权成本 PBV（近似）
−4.50% 0.5
−0.50% 0.9
5.50% 2.5
10.50% 4.0
15.50% 5.5
20.50% 7.0
25.50% 8.5
30.50% 9.5

核心洞察：当 ROE = 股权成本时，PBV = 1。超额回报每增加 5 个百分点，PBV 大约增加 1.5-2.0。

4.3 ROE 的决定因素：Porter 五力框架

ROE 与股权成本的差异衡量了公司获取超额回报的能力。这种能力从何而来？Michael Porter 的五力框架提供了分析框架：

五种竞争力量：

  1. 新进入者威胁

    • 规模经济
    • 成本优势
    • 资本要求
    • 产品差异化
    • 分销渠道
    • 政府和法律壁垒

  2. 供应商议价能力

  3. 买方议价能力

    • 购买量占总成本的比例
    • 买方盈利能力
    • 产品对买方的重要性

  4. 替代品威胁

    • 买方替代倾向
    • 替代品的相对价格表现

  5. 行业内竞争

    • 集中度
    • 产品差异化
    • 产能过剩
    • 固定成本/可变成本比率
注记护城河（Moat）概念

价值投资者将持续的竞争优势称为”护城河”。在本章的框架中，护城河的强度由两个因素决定：

  1. ROE 的水平：护城河越强，ROE 越高
  2. ROE 的持续性：护城河越宽，高 ROE 能维持越久

5 PBV 的应用：从市场到个股

5.1 市场层面：PBV 与 ROE 的历史关系

图 19.4 展示了 S&P 500 从 1980 年到 2023 年的 PBV 和 ROE 变化：

关键观察：

时期 趋势
1980-2000 PBV 和 ROE 同步上升（科技股涌入）
2001-2010 PBV 和 ROE 同步下降
2011-2021 PBV 和 ROE 再次上升（低利率贡献）

解读：市场整体的 PBV 由企业整体的超额回报能力决定。低利率降低了股权成本，从而推高了 PBV。

5.2 行业内比较：矩阵法与回归法

矩阵法：

将公司按 PBV 和 ROE（或超额回报）分类：

低超额回报 高超额回报
高 PBV 高估 合理定价
低 PBV 合理定价 低估

值得关注的是不匹配的公司：

  • 低 PBV + 高 ROE → 潜在低估
  • 高 PBV + 低 ROE → 潜在高估

回归法：

可以用回归模型量化 PBV 与 ROE 的关系：

\[ \text{PBV} = a + b \times \text{ROE} \]

如果关系显著，可以用回归预测每家公司的”合理” PBV，然后比较实际 PBV 与预测 PBV。

5.3 欧洲服装公司案例分析

5.6 时间序列比较：IBM 的兴衰

IBM 提供了 ROE 与 PBV 关系的经典案例：

时期 ROE PBV 事件
1983 25% 3.0× 道琼斯最高 PBV 之一
1992-1993 负值 ~1.0× ROE 崩溃
1999 高 9.0× Gerstner 改革成功
2001-2010 持续高 高 维持高 ROE 和高 PBV

投资启示：在低点买入 IBM 的投资者，押注的是 ROE 的改善。随着公司从”低 PBV/低 ROE”象限移动到”高 PBV/高 ROE”象限，股价大幅上涨。

提示当前 ROE vs 预期 ROE

在所有比较中，我们使用的是当前 ROE。但市场定价反映的是预期未来 ROE。

两种调整方法：

  1. 历史平均法：使用 3-5 年平均 ROE，减少波动影响
  2. 均值回归法：将当前 ROE 向行业平均推移，反映竞争压力

例如，一家软件公司当前 ROE 35%，行业平均 20%。预测 ROE 可以是两者的加权平均，权重取决于竞争优势的持续性。

5.7 有形账面价值与非现金 PBV

一些价值投资者偏好使用有形账面价值：

\[ \text{有形账面价值} = \text{账面价值} - \text{商誉} \]

相应地：

\[ \text{有形 ROE} = \frac{\text{净利润}}{\text{账面价值} - \text{商誉}} \]

优点：剔除了收购中可能的溢价支付

注意：商誉反映了控制权溢价、协同效应，也可能包含过度支付。持续过度支付收购的公司，其股权实际上价值更低。

6.2 Benjamin Graham 的选股标准

本杰明·格雷厄姆在其经典著作中将”股价低于账面价值的三分之二”作为选股标准之一。

但本章的分析表明，单纯的低 PBV 不足以识别低估。真正的低估是：

\[ \text{低估} = \text{低 PBV} + \text{高 ROE（预期）} \]

6.3 PBV 作为风险代理？

Fama 和 French (1992) 提出，低 PBV 股票的高回报可能是风险补偿，而非定价错误。

反驳：

  • 低 PBV 股票组合的杠杆和盈利波动性与高 PBV 组合相似
  • 没有明确证据表明低 PBV 股票面临更高的系统性风险

这仍然是学术界争论的问题。

7 企业价值/账面资本比率（Value-to-Book）

7.1 定义

企业价值/账面资本比率是 PBV 的公司价值版本：

\[ \text{企业价值/账面资本} = \frac{\text{股权市值} + \text{债务市值}}{\text{股权账面价值} + \text{债务账面价值}} \]

如果使用企业价值（EV）作为分子（扣除现金），分母也应相应调整为投入资本（Invested Capital）：

\[ \text{EV/IC} = \frac{\text{股权市值} + \text{债务市值} - \text{现金}}{\text{股权账面} + \text{债务账面} - \text{现金}} \]

警告一致性要求

两种常见错误：

  1. 使用总资产作为分母：总资产包含流动负债，会低估比率
  2. 分子扣现金但分母不扣：导致比率偏低

正确做法：分子分母对现金的处理保持一致。

7.2 分布特征

2024 年 1 月 EV/IC 的分布：

统计量 美国 全球
中位数 1.97 1.54

优点：负投入资本的公司比负账面权益的公司更少，样本损失更小。

7.3 从 DCF 推导 EV/IC

从 FCFF 估值模型出发，对于稳定增长公司：

\[ \text{企业价值} = \frac{\text{FCFF}_1}{k_c - g} = \frac{\text{EBIT}(1-t) \times (1 - \text{再投资率})}{k_c - g} \]

定义 ROIC = EBIT(1−t) / 投入资本，可得：

\[ \frac{\text{EV}}{\text{IC}} = \frac{\text{ROIC} \times (1 - \text{再投资率})}{k_c - g} \]

利用 \(\text{再投资率} = g / \text{ROIC}\)，简化为：

\[ \boxed{\frac{\text{EV}}{\text{IC}} = \frac{\text{ROIC} - g}{k_c - g}} \]

这与 PBV 的公式结构完全相同，只是：

PBV EV/IC
ROE ROIC
股权成本 \(k_e\) 资本成本 \(k_c\)
派息率 1 − 再投资率

8 托宾 Q（Tobin’s Q）

8.1 定义

诺贝尔经济学奖得主 James Tobin 提出的指标：

\[ \text{Tobin's Q} = \frac{\text{资产市值}}{\text{资产重置成本}} \]

理论基础：

  • 如果 Q < 1，市场认为公司的资产不值其重置成本——可能管理不善
  • 如果 Q > 1，公司从资产中创造的价值超过重置成本——效率优势

与超额回报的关系：

当资产回报率 = 资本成本时，投资 NPV = 0，资产市值 = 重置成本，Q = 1。

8.2 实际应用的局限

主要问题：

  1. 重置成本难以估计：特别是非标准化资产
  2. 数据获取困难：需要大量额外信息
  3. 通胀和技术变化：
    • 通胀时期，重置成本上升，Q 下降
    • 技术进步使重置成本下降，Q 上升

实践中的简化：

\[ \text{Tobin's Q} \approx \frac{\text{股权市值} + \text{债务市值}}{\text{总资产账面价值}} \]

这实际上就是 EV/IC 的变体。

8.3 应用场景

最适合：

  • 成熟公司，资产大部分已到位
  • 可以合理估计重置成本的行业（如钢铁、公用事业）
  • 作为管理效率的衡量指标

不适合：

  • 高增长公司（市值包含大量增长期权）
  • 无形资产为主的公司

学术研究发现（Lang, Stulz & Walkling, 1991）：

  • 低 Q 公司更可能被收购进行重组
  • 高 Q 公司作为收购方时，股东获益更多

9 总结

重要核心要点
  • PBV 的本质：市场对公司创造超额股权回报能力的定价
  • 核心公式：\(\text{PBV} = \frac{\text{ROE} - g}{k_e - g}\)
  • 临界点：当 ROE = 股权成本时，PBV = 1
  • 识别低估：低 PBV + 高 ROE = 潜在低估；高 PBV + 低 ROE = 潜在高估
  • EV/IC：公司价值版本的 PBV，适合高杠杆或负账面权益公司
  • Tobin’s Q：理论优雅但实践困难，更多用于学术研究

本章的核心信息可以概括为一句话：不要因为 PBV 低就认为股票便宜——只有当 ROE 高于股权成本时，低 PBV 才是真正的便宜。

这与 PE 分析（第 18 章）的逻辑是一致的：单看倍数高低没有意义，必须结合其基本面决定因素——对于 PBV，这个决定因素就是 ROE。
