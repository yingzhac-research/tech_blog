1 从一个问题开始

假设你需要为两家公司估值：一家是美国的成熟科技公司苹果，另一家是巴西的新兴电商平台。直觉告诉我们，投资巴西公司应该要求更高的回报——但高多少？5%？10%？这个”额外要求的回报”从何而来？

更基础的问题是：当我们说”投资者要求10%的回报”时，这个10%是怎么构成的？

答案就藏在本章要讨论的几个关键概念中：无风险利率（Risk-free Rate）、股权风险溢价（Equity Risk Premium, ERP）、以及国家风险溢价（Country Risk Premium）。这些参数看似简单，却是整个估值体系中最具争议、也最容易出错的地方。

Important为什么这些参数如此重要？

在DCF估值中，折现率的微小变化会导致估值的巨大波动。假设一家公司的永续现金流是100亿，如果折现率从10%变成9%，估值就从1000亿跳到1111亿——差了111亿！而折现率的核心组成部分，就是本章讨论的无风险利率和风险溢价。

2 无风险利率：一切的起点

2.1 什么是真正的”无风险”？

我们先思考一个看似简单的问题：什么样的投资才是真正无风险的？

表面上，“无风险”意味着”确定能拿回钱”。但Damodaran指出，真正的无风险投资需要满足两个条件：

  1. 没有违约风险（No Default Risk）：发行方必须100%能还钱
  2. 没有再投资风险（No Reinvestment Risk）：期间产生的现金流再投资时，收益率必须是确定的

第一个条件比较直观——政府可以印钱还债，所以政府债券通常被认为没有违约风险。但第二个条件常被忽略：如果你买了一张10年期债券，期间每年收到的利息，当你再投资时的利率是不确定的。

Note零息债券：唯一真正无风险的工具

严格来说，只有零息政府债券（Zero-coupon Government Bond）才是真正的无风险投资。因为它没有中间的利息支付，持有到期时的回报是完全确定的。

但在实践中，我们通常用期限匹配的政府债券收益率作为无风险利率的近似。

2.2 实践中的关键原则：期限匹配

这里有一个至关重要的原则：无风险利率的期限必须与被折现的现金流期限匹配。

为什么？让我们用一个例子来说明。假设你要估算一家公司未来10年的现金流，如果用1年期国债利率（比如4%）来折现，而10年期国债利率是5%，你就犯了一个隐蔽但严重的错误：

  • 用4%折现意味着你假设投资者可以持续10年都以4%的利率滚动投资
  • 但实际上，未来的短期利率是不确定的
  • 10年期国债的5%收益率已经包含了对这种不确定性的定价
现金流期限 应使用的无风险利率
1年 1年期国债利率
5年 5年期国债利率
10年以上（如永续） 10年期或30年期国债利率
Tip实务建议

对于大多数DCF估值，由于我们估算的是长期现金流（通常包含一个永续终值），应该使用10年期或更长期限的政府债券利率作为无风险利率。

2024年12月，美国10年期国债收益率约为4.5%，这是估值美国公司时的起点。

2.3 当没有”无风险”政府时怎么办？

美国政府债券被认为是无风险的，因为美国政府可以印美元。但如果你在巴西、印度或阿根廷做估值呢？这些国家的政府债券可能存在违约风险。

Damodaran提供了几种解决方案：

方法一：使用本币计价的发达国家债券

如果你用美元做估值，可以直接用美国国债利率。但要确保所有的现金流预测也是美元计价的。

方法二：用主权CDS利差调整

政府债券收益率 - 主权CDS利差 ≈ 无风险利率

主权CDS（Credit Default Swap）利差反映了市场对该国违约风险的定价。例如：

  • 巴西10年期政府债券收益率：12%
  • 巴西主权CDS利差：2.5%
  • 巴西的雷亚尔无风险利率 ≈ 12% - 2.5% = 9.5%

方法三：通过通胀率转换

如果你知道美元无风险利率（4.5%），可以通过预期通胀率差异转换为其他货币：

\[ (1 + r_{本币}) = (1 + r_{美元}) \times \frac{1 + \pi_{本币}}{1 + \pi_{美元}} \]

其中 \(\pi\) 代表预期通胀率。

Warning货币一致性原则

这是估值中最容易犯的错误之一：现金流的货币必须与折现率的货币一致。

如果你用美元预测现金流，就用美元无风险利率；如果用人民币预测现金流，就用人民币无风险利率。混用会导致严重的估值错误。

3 股权风险溢价：投资者要求的额外回报

3.1 直觉：为什么股票要比国债赚更多？

无风险利率回答了”把钱借给最安全的借款人能赚多少”的问题。但投资股票比买国债风险大得多——公司可能破产、利润可能下滑、股价可能暴跌。

因此，理性的投资者会要求额外的回报来补偿这些风险。这个额外回报就是股权风险溢价（Equity Risk Premium, ERP）。

\[ E(R_{股票}) = R_f + ERP \]

其中：

  • \(E(R_{股票})\)：投资股票的期望回报率
  • \(R_f\)：无风险利率
  • \(ERP\)：股权风险溢价

关键问题是：ERP应该是多少？这是金融学中争议最大的问题之一。

3.2 方法一：历史风险溢价

最直观的方法是看历史：过去股票比国债多赚了多少？

以美国市场为例，Damodaran统计了1928-2023年的数据：

比较基准 算术平均 几何平均
股票 vs 短期国债 8.4% 6.6%
股票 vs 长期国债 6.2% 4.6%

但这个方法有几个重要问题：

问题1：用什么时间段？

  • 用过去20年？50年？100年？
  • 不同时期的结果差异巨大
  • 短期数据波动大，长期数据可能不代表未来

问题2：算术平均还是几何平均？

这是一个关键的技术问题。假设某股票第一年涨100%（从100涨到200），第二年跌50%（从200跌回100）：

  • 算术平均回报 = (100% + (-50%)) / 2 = 25%
  • 几何平均回报 = \(\sqrt{1 \times 1} - 1\) = 0%

哪个对？几何平均反映了真实的累计收益，而算术平均是对未来单期回报的无偏估计。

Note选择建议
  • 如果你做的是单期估值（比如预测明年的回报），用算术平均
  • 如果你做的是多期DCF估值，几何平均更合适
  • 实务中，很多人取两者的平均作为折中

问题3：比较基准是短期还是长期国债？

  • 短期国债（T-bills）：几乎没有利率风险，但与长期投资期限不匹配
  • 长期国债（T-bonds）：期限匹配，但本身有利率风险

如前所述，由于DCF估值涉及长期现金流，应该用长期国债作为比较基准。

3.3 方法二：隐含股权风险溢价

历史法的根本问题是：过去不代表未来。市场结构、投资者构成、风险偏好都在变化。

Damodaran推荐的替代方法是隐含股权风险溢价：从当前的市场价格反推投资者要求的回报率。

核心思路是：如果我们知道市场的当前价格、预期股息和预期增长率，就可以用Gordon增长模型反推折现率：

\[ 价格 = \frac{预期股息}{折现率 - 增长率} \]

变形后：

\[ 折现率 = \frac{预期股息}{价格} + 增长率 \]

用这个折现率减去无风险利率，就得到隐含的股权风险溢价。

具体例子：2024年1月的S&P 500

让我们用实际数据来计算：

  • S&P 500指数：4,770点
  • 过去12个月的股息 + 回购：约205点
  • 分析师预测未来5年盈利增长率：约8%
  • 5年后假设进入稳态增长：4%
  • 10年期国债收益率：4.5%

通过迭代求解（使现金流现值等于4,770），可以得到隐含的期望回报率约为10.2%。

因此，隐含ERP = 10.2% - 4.5% = 5.7%

Important隐含法的优势
  1. 前瞻性：反映当前市场对风险的定价，而非历史平均
  2. 动态更新：可以随市场变化实时调整
  3. 内部一致：与当前市场价格保持一致

Damodaran每月在他的网站上更新隐含ERP的估算，这是实务中非常有价值的参考。

3.4 两种方法的比较

维度 历史法 隐含法
数据来源 历史回报数据 当前市场价格
时间导向 后视镜 前瞻性
稳定性 较稳定（除非改变时间窗口） 随市场波动
适用场景 学术研究、长期趋势 实时估值、市场时机
2024年估计值 约4.5-6.5%（取决于参数） 约5.5-6%（美国市场）

4 国家风险溢价：新兴市场的额外风险

4.1 为什么需要国家风险溢价？

到目前为止，我们讨论的ERP主要针对美国这样的成熟市场。但如果你要估值一家印度公司或巴西公司呢？

这些新兴市场面临额外的风险：

  • 政治风险：政权更迭、政策不确定性
  • 经济风险：货币波动、通胀不稳定
  • 法律风险：产权保护不足、合同执行困难
  • 流动性风险：市场深度不足，交易成本高

投资者会要求额外的溢价来补偿这些风险。问题是：这个额外溢价应该是多少？

4.2 估算国家风险溢价的三种方法

方法一：主权债券违约利差法

最简单的方法是看该国政府债券相对于美国国债的利差：

\[ 国家风险溢价 = 该国美元债券收益率 - 美国国债收益率 \]

例如，如果巴西发行的美元计价债券收益率是7%，美国国债是4.5%，则巴西的国家风险溢价 = 2.5%。

但这个方法有个问题：它测量的是债券市场的风险溢价，而股票的风险通常比债券大。

方法二：相对波动率调整法

为了调整股票相对于债券的更高波动性，可以用两个市场的波动率比例来放大：

\[ 国家股权风险溢价 = 国家债券利差 \times \frac{\sigma_{股票}}{\sigma_{债券}} \]

其中 \(\sigma\) 代表波动率。

假设新兴市场股票的波动率是债券的1.5倍（这是一个典型的比例），则：

巴西股权风险溢价 = 2.5% × 1.5 = 3.75%

方法三：综合评级法

Damodaran建议使用国家信用评级来估算风险溢价，并提供了一个对照表。例如（2024年数据）：

国家评级（穆迪） 典型国家 额外风险溢价
Aaa 德国、新加坡 0%
Aa1/Aa2 日本、韩国 0.5-0.7%
A1/A2 中国、波兰 1.0-1.5%
Baa1/Baa2 印度、墨西哥 2.0-2.5%
Ba1/Ba2 巴西、印尼 3.0-4.0%
B1及以下 阿根廷、土耳其 5.0%+
Tip实务建议：完整的新兴市场折现率

假设你要估值一家印度公司（评级约Baa3）：

  1. 美元无风险利率：4.5%（10年期美国国债）
  2. 成熟市场ERP：5.5%（隐含法估算）
  3. 印度国家风险溢价：约2.5%

印度公司的股权成本 = 4.5% + 5.5% + 2.5% = 12.5%

（这只是Beta=1的基准，实际还需要乘以公司的Beta系数）

4.3 一个重要的细微差别：公司风险敞口

并非所有位于新兴市场的公司都面临同样程度的国家风险：

  • 一家巴西出口商：收入来自全球，可能只承担部分巴西风险
  • 一家巴西本地零售商：收入完全来自巴西，承担全部巴西风险
  • 一家美国公司在巴西的子公司：取决于业务结构

Damodaran建议根据公司的收入来源地理分布来分配国家风险溢价，而不是简单地看公司注册地。

5 债券违约利差：债务成本的风险溢价

5.1 从无风险到有风险的债务

前面讨论的都是股权成本。但公司也会借债，债务成本怎么确定？

核心公式很简单：

\[ 债务成本 = 无风险利率 + 违约利差 \]

违约利差（Default Spread）反映了债权人要求的额外回报，以补偿借款人可能无法偿还的风险。

5.2 违约利差的决定因素

违约利差主要取决于公司的信用评级。评级越低，违约风险越高，利差越大：

评级（标普） 典型违约利差（2024年）
AAA 0.5-0.7%
AA 0.8-1.0%
A 1.0-1.5%
BBB 1.5-2.5%
BB 3.0-4.0%
B 5.0-6.0%
CCC及以下 8.0%+
Warning违约利差会随市场周期变化

上面的数字是”正常”市场环境下的典型值。但在市场恐慌时期（如2008年金融危机、2020年3月），违约利差会急剧扩大：

  • 2007年：BBB级债券利差约1.5%
  • 2008年底：同样评级的利差飙升到超过6%
  • 2024年：回到约2%

因此，使用违约利差时，既要考虑公司的评级，也要考虑当前的市场环境。

5.3 如果公司没有评级怎么办？

很多公司（尤其是非上市公司）没有正式的信用评级。这时可以用合成评级（Synthetic Rating）：

  1. 计算公司的利息覆盖率（Interest Coverage Ratio）= EBIT / 利息费用
  2. 根据利息覆盖率查表得到对应的合成评级
  3. 用合成评级查找对应的违约利差
利息覆盖率 合成评级 典型违约利差
> 12.5 AAA 0.5%
9.5-12.5 AA 0.8%
7.5-9.5 A+ 1.0%
6.0-7.5 A 1.2%
4.5-6.0 A- 1.4%
4.0-4.5 BBB 1.8%
3.5-4.0 BB+ 2.5%
3.0-3.5 BB 3.0%
2.5-3.0 B+ 4.0%
2.0-2.5 B 5.0%
< 2.0 CCC 8.0%

6 把所有部分组合在一起

让我们用一个完整的例子来演示如何使用本章的所有概念。

案例：估算一家巴西钢铁公司的折现率

假设条件： - 公司在巴西运营，60%收入来自巴西，40%出口到全球 - 公司Beta系数：1.2 - 公司信用评级：BB+ - 债务/资本比率：40% - 税率：34%

步骤1：确定无风险利率 美元10年期国债收益率：4.5%

步骤2：计算股权成本

基础ERP（成熟市场）：5.5%

国家风险溢价： - 巴西基础溢价：3.5% - 根据收入来源调整：3.5% × 60% = 2.1%

总ERP = 5.5% + 2.1% = 7.6%

股权成本 = 4.5% + 1.2 × 7.6% = 13.6%

步骤3：计算债务成本

BB+评级违约利差：约2.5%

税前债务成本 = 4.5% + 2.5% = 7.0% 税后债务成本 = 7.0% × (1 - 34%) = 4.6%

步骤4：计算WACC

\[ WACC = 60\% \times 13.6\% + 40\% \times 4.6\% = 8.2\% + 1.8\% = \textbf{10.0\%} \]

Important小结：折现率的完整构成

\[ \text{股权成本} = R_f + \beta \times (ERP_{成熟市场} + CRP) \]

\[ \text{债务成本} = R_f + \text{违约利差} \]

\[ WACC = \frac{E}{V} \times R_e + \frac{D}{V} \times R_d \times (1-T) \]

每个输入参数的来源：

  • \(R_f\)：政府债券收益率（期限匹配）
  • \(ERP\)：历史法或隐含法估算
  • \(CRP\)：主权利差或评级查表
  • 违约利差：公司评级或合成评级

7 本章总结

本章讨论了估值中最基础也最关键的几个输入参数：

  1. 无风险利率是一切的起点，必须满足无违约风险和无再投资风险，实践中用期限匹配的政府债券利率

  2. 股权风险溢价可以用历史法或隐含法估算，隐含法更具前瞻性，当前美国市场的ERP约为5.5-6%

  3. 国家风险溢价补偿新兴市场的额外风险，可以通过主权利差、相对波动率或评级表来估算

  4. 违约利差决定债务成本，取决于公司信用评级和市场环境

  5. 货币一致性是容易犯错但绝对不能犯错的原则

8 思考题

Note问题1

为什么Damodaran更推荐用隐含ERP而不是历史ERP？如果市场处于泡沫期（如2000年互联网泡沫），隐含ERP会有什么问题？

Note问题2

假设你要估值一家中国公司，该公司80%的收入来自美国市场。在计算国家风险溢价时，你会如何处理？

Note问题3

在利率上升周期中（如2022-2023年），无风险利率上升对ERP的估算有什么影响？对公司估值的整体影响是什么？

Note问题4

为什么计算债务成本时要用税后利率？如果一家公司连续亏损，无法享受利息抵税，应该如何调整WACC计算？

Note问题5

违约利差在经济周期的不同阶段变化很大。作为估值分析师，你应该使用当前的违约利差，还是某种”正常化”的利差？为什么？

## 从一个问题开始

假设你需要为两家公司估值：一家是美国的成熟科技公司苹果，另一家是巴西的新兴电商平台。直觉告诉我们，投资巴西公司应该要求更高的回报——但高多少？5%？10%？这个"额外要求的回报"从何而来？

更基础的问题是：当我们说"投资者要求10%的回报"时，这个10%是怎么构成的？

答案就藏在本章要讨论的几个关键概念中：**无风险利率**（Risk-free Rate）、**股权风险溢价**（Equity Risk Premium, ERP）、以及**国家风险溢价**（Country Risk Premium）。这些参数看似简单，却是整个估值体系中最具争议、也最容易出错的地方。

::: {.callout-important}
## 为什么这些参数如此重要？

在DCF估值中，折现率的微小变化会导致估值的巨大波动。假设一家公司的永续现金流是100亿，如果折现率从10%变成9%，估值就从1000亿跳到1111亿——差了111亿！而折现率的核心组成部分，就是本章讨论的无风险利率和风险溢价。
:::

## 无风险利率：一切的起点

### 什么是真正的"无风险"？

我们先思考一个看似简单的问题：什么样的投资才是真正无风险的？

表面上，"无风险"意味着"确定能拿回钱"。但Damodaran指出，真正的无风险投资需要满足两个条件：

1. **没有违约风险**（No Default Risk）：发行方必须100%能还钱
2. **没有再投资风险**（No Reinvestment Risk）：期间产生的现金流再投资时，收益率必须是确定的

第一个条件比较直观——政府可以印钱还债，所以政府债券通常被认为没有违约风险。但第二个条件常被忽略：如果你买了一张10年期债券，期间每年收到的利息，当你再投资时的利率是不确定的。

::: {.callout-note}
## 零息债券：唯一真正无风险的工具

严格来说，只有**零息政府债券**（Zero-coupon Government Bond）才是真正的无风险投资。因为它没有中间的利息支付，持有到期时的回报是完全确定的。

但在实践中，我们通常用**期限匹配的政府债券收益率**作为无风险利率的近似。
:::

### 实践中的关键原则：期限匹配

这里有一个至关重要的原则：**无风险利率的期限必须与被折现的现金流期限匹配**。

为什么？让我们用一个例子来说明。假设你要估算一家公司未来10年的现金流，如果用1年期国债利率（比如4%）来折现，而10年期国债利率是5%，你就犯了一个隐蔽但严重的错误：

- 用4%折现意味着你假设投资者可以持续10年都以4%的利率滚动投资
- 但实际上，未来的短期利率是不确定的
- 10年期国债的5%收益率已经包含了对这种不确定性的定价

| 现金流期限 | 应使用的无风险利率 |
|------------|-------------------|
| 1年 | 1年期国债利率 |
| 5年 | 5年期国债利率 |
| 10年以上（如永续） | 10年期或30年期国债利率 |

::: {.callout-tip}
## 实务建议

对于大多数DCF估值，由于我们估算的是长期现金流（通常包含一个永续终值），应该使用**10年期或更长期限的政府债券利率**作为无风险利率。

2024年12月，美国10年期国债收益率约为4.5%，这是估值美国公司时的起点。
:::

### 当没有"无风险"政府时怎么办？

美国政府债券被认为是无风险的，因为美国政府可以印美元。但如果你在巴西、印度或阿根廷做估值呢？这些国家的政府债券可能存在违约风险。

Damodaran提供了几种解决方案：

**方法一：使用本币计价的发达国家债券**

如果你用美元做估值，可以直接用美国国债利率。但要确保所有的现金流预测也是美元计价的。

**方法二：用主权CDS利差调整**

政府债券收益率 - 主权CDS利差 ≈ 无风险利率

主权CDS（Credit Default Swap）利差反映了市场对该国违约风险的定价。例如：

- 巴西10年期政府债券收益率：12%
- 巴西主权CDS利差：2.5%
- 巴西的雷亚尔无风险利率 ≈ 12% - 2.5% = 9.5%

**方法三：通过通胀率转换**

如果你知道美元无风险利率（4.5%），可以通过预期通胀率差异转换为其他货币：

$$
(1 + r_{本币}) = (1 + r_{美元}) \times \frac{1 + \pi_{本币}}{1 + \pi_{美元}}
$$

其中 $\pi$ 代表预期通胀率。

::: {.callout-warning}
## 货币一致性原则

这是估值中最容易犯的错误之一：**现金流的货币必须与折现率的货币一致**。

如果你用美元预测现金流，就用美元无风险利率；如果用人民币预测现金流，就用人民币无风险利率。混用会导致严重的估值错误。
:::

## 股权风险溢价：投资者要求的额外回报

### 直觉：为什么股票要比国债赚更多？

无风险利率回答了"把钱借给最安全的借款人能赚多少"的问题。但投资股票比买国债风险大得多——公司可能破产、利润可能下滑、股价可能暴跌。

因此，理性的投资者会要求**额外的回报**来补偿这些风险。这个额外回报就是**股权风险溢价**（Equity Risk Premium, ERP）。

$$
E(R_{股票}) = R_f + ERP
$$

其中：

- $E(R_{股票})$：投资股票的期望回报率
- $R_f$：无风险利率
- $ERP$：股权风险溢价

关键问题是：ERP应该是多少？这是金融学中争议最大的问题之一。

### 方法一：历史风险溢价

最直观的方法是看历史：过去股票比国债多赚了多少？

以美国市场为例，Damodaran统计了1928-2023年的数据：

| 比较基准 | 算术平均 | 几何平均 |
|----------|----------|----------|
| 股票 vs 短期国债 | 8.4% | 6.6% |
| 股票 vs 长期国债 | 6.2% | 4.6% |

但这个方法有几个重要问题：

**问题1：用什么时间段？**

- 用过去20年？50年？100年？
- 不同时期的结果差异巨大
- 短期数据波动大，长期数据可能不代表未来

**问题2：算术平均还是几何平均？**

这是一个关键的技术问题。假设某股票第一年涨100%（从100涨到200），第二年跌50%（从200跌回100）：

- 算术平均回报 = (100% + (-50%)) / 2 = 25%
- 几何平均回报 = $\sqrt{1 \times 1} - 1$ = 0%

哪个对？**几何平均**反映了真实的累计收益，而**算术平均**是对未来单期回报的无偏估计。

::: {.callout-note}
## 选择建议

- 如果你做的是**单期**估值（比如预测明年的回报），用算术平均
- 如果你做的是**多期**DCF估值，几何平均更合适
- 实务中，很多人取两者的平均作为折中
:::

**问题3：比较基准是短期还是长期国债？**

- 短期国债（T-bills）：几乎没有利率风险，但与长期投资期限不匹配
- 长期国债（T-bonds）：期限匹配，但本身有利率风险

如前所述，由于DCF估值涉及长期现金流，**应该用长期国债作为比较基准**。

### 方法二：隐含股权风险溢价

历史法的根本问题是：**过去不代表未来**。市场结构、投资者构成、风险偏好都在变化。

Damodaran推荐的替代方法是**隐含股权风险溢价**：从当前的市场价格反推投资者要求的回报率。

核心思路是：如果我们知道市场的当前价格、预期股息和预期增长率，就可以用Gordon增长模型反推折现率：

$$
价格 = \frac{预期股息}{折现率 - 增长率}
$$

变形后：

$$
折现率 = \frac{预期股息}{价格} + 增长率
$$

用这个折现率减去无风险利率，就得到**隐含的股权风险溢价**。

**具体例子：2024年1月的S&P 500**

让我们用实际数据来计算：

- S&P 500指数：4,770点
- 过去12个月的股息 + 回购：约205点
- 分析师预测未来5年盈利增长率：约8%
- 5年后假设进入稳态增长：4%
- 10年期国债收益率：4.5%

通过迭代求解（使现金流现值等于4,770），可以得到隐含的期望回报率约为10.2%。

因此，隐含ERP = 10.2% - 4.5% = **5.7%**

::: {.callout-important}
## 隐含法的优势

1. **前瞻性**：反映当前市场对风险的定价，而非历史平均
2. **动态更新**：可以随市场变化实时调整
3. **内部一致**：与当前市场价格保持一致

Damodaran每月在他的网站上更新隐含ERP的估算，这是实务中非常有价值的参考。
:::

### 两种方法的比较

| 维度 | 历史法 | 隐含法 |
|------|--------|--------|
| 数据来源 | 历史回报数据 | 当前市场价格 |
| 时间导向 | 后视镜 | 前瞻性 |
| 稳定性 | 较稳定（除非改变时间窗口） | 随市场波动 |
| 适用场景 | 学术研究、长期趋势 | 实时估值、市场时机 |
| 2024年估计值 | 约4.5-6.5%（取决于参数） | 约5.5-6%（美国市场） |

## 国家风险溢价：新兴市场的额外风险

### 为什么需要国家风险溢价？

到目前为止，我们讨论的ERP主要针对美国这样的成熟市场。但如果你要估值一家印度公司或巴西公司呢？

这些新兴市场面临额外的风险：

- **政治风险**：政权更迭、政策不确定性
- **经济风险**：货币波动、通胀不稳定
- **法律风险**：产权保护不足、合同执行困难
- **流动性风险**：市场深度不足，交易成本高

投资者会要求**额外的溢价**来补偿这些风险。问题是：这个额外溢价应该是多少？

### 估算国家风险溢价的三种方法

**方法一：主权债券违约利差法**

最简单的方法是看该国政府债券相对于美国国债的利差：

$$
国家风险溢价 = 该国美元债券收益率 - 美国国债收益率
$$

例如，如果巴西发行的美元计价债券收益率是7%，美国国债是4.5%，则巴西的国家风险溢价 = 2.5%。

但这个方法有个问题：它测量的是**债券**市场的风险溢价，而股票的风险通常比债券大。

**方法二：相对波动率调整法**

为了调整股票相对于债券的更高波动性，可以用两个市场的波动率比例来放大：

$$
国家股权风险溢价 = 国家债券利差 \times \frac{\sigma_{股票}}{\sigma_{债券}}
$$

其中 $\sigma$ 代表波动率。

假设新兴市场股票的波动率是债券的1.5倍（这是一个典型的比例），则：

巴西股权风险溢价 = 2.5% × 1.5 = **3.75%**

**方法三：综合评级法**

Damodaran建议使用国家信用评级来估算风险溢价，并提供了一个对照表。例如（2024年数据）：

| 国家评级（穆迪） | 典型国家 | 额外风险溢价 |
|-----------------|----------|-------------|
| Aaa | 德国、新加坡 | 0% |
| Aa1/Aa2 | 日本、韩国 | 0.5-0.7% |
| A1/A2 | 中国、波兰 | 1.0-1.5% |
| Baa1/Baa2 | 印度、墨西哥 | 2.0-2.5% |
| Ba1/Ba2 | 巴西、印尼 | 3.0-4.0% |
| B1及以下 | 阿根廷、土耳其 | 5.0%+ |

::: {.callout-tip}
## 实务建议：完整的新兴市场折现率

假设你要估值一家印度公司（评级约Baa3）：

1. 美元无风险利率：4.5%（10年期美国国债）
2. 成熟市场ERP：5.5%（隐含法估算）
3. 印度国家风险溢价：约2.5%

**印度公司的股权成本** = 4.5% + 5.5% + 2.5% = **12.5%**

（这只是Beta=1的基准，实际还需要乘以公司的Beta系数）
:::

### 一个重要的细微差别：公司风险敞口

并非所有位于新兴市场的公司都面临同样程度的国家风险：

- 一家**巴西出口商**：收入来自全球，可能只承担部分巴西风险
- 一家**巴西本地零售商**：收入完全来自巴西，承担全部巴西风险
- 一家**美国公司在巴西的子公司**：取决于业务结构

Damodaran建议根据公司的**收入来源地理分布**来分配国家风险溢价，而不是简单地看公司注册地。

## 债券违约利差：债务成本的风险溢价

### 从无风险到有风险的债务

前面讨论的都是股权成本。但公司也会借债，债务成本怎么确定？

核心公式很简单：

$$
债务成本 = 无风险利率 + 违约利差
$$

**违约利差**（Default Spread）反映了债权人要求的额外回报，以补偿借款人可能无法偿还的风险。

### 违约利差的决定因素

违约利差主要取决于公司的**信用评级**。评级越低，违约风险越高，利差越大：

| 评级（标普） | 典型违约利差（2024年） |
|-------------|----------------------|
| AAA | 0.5-0.7% |
| AA | 0.8-1.0% |
| A | 1.0-1.5% |
| BBB | 1.5-2.5% |
| BB | 3.0-4.0% |
| B | 5.0-6.0% |
| CCC及以下 | 8.0%+ |

::: {.callout-warning}
## 违约利差会随市场周期变化

上面的数字是"正常"市场环境下的典型值。但在市场恐慌时期（如2008年金融危机、2020年3月），违约利差会急剧扩大：

- 2007年：BBB级债券利差约1.5%
- 2008年底：同样评级的利差飙升到超过6%
- 2024年：回到约2%

因此，使用违约利差时，既要考虑公司的评级，也要考虑当前的市场环境。
:::

### 如果公司没有评级怎么办？

很多公司（尤其是非上市公司）没有正式的信用评级。这时可以用**合成评级**（Synthetic Rating）：

1. 计算公司的**利息覆盖率**（Interest Coverage Ratio）= EBIT / 利息费用
2. 根据利息覆盖率查表得到对应的合成评级
3. 用合成评级查找对应的违约利差

| 利息覆盖率 | 合成评级 | 典型违约利差 |
|------------|----------|-------------|
| > 12.5 | AAA | 0.5% |
| 9.5-12.5 | AA | 0.8% |
| 7.5-9.5 | A+ | 1.0% |
| 6.0-7.5 | A | 1.2% |
| 4.5-6.0 | A- | 1.4% |
| 4.0-4.5 | BBB | 1.8% |
| 3.5-4.0 | BB+ | 2.5% |
| 3.0-3.5 | BB | 3.0% |
| 2.5-3.0 | B+ | 4.0% |
| 2.0-2.5 | B | 5.0% |
| < 2.0 | CCC | 8.0% |

## 把所有部分组合在一起

让我们用一个完整的例子来演示如何使用本章的所有概念。

**案例：估算一家巴西钢铁公司的折现率**

假设条件：
- 公司在巴西运营，60%收入来自巴西，40%出口到全球
- 公司Beta系数：1.2
- 公司信用评级：BB+
- 债务/资本比率：40%
- 税率：34%

**步骤1：确定无风险利率**
美元10年期国债收益率：4.5%

**步骤2：计算股权成本**

基础ERP（成熟市场）：5.5%

国家风险溢价：
- 巴西基础溢价：3.5%
- 根据收入来源调整：3.5% × 60% = 2.1%

总ERP = 5.5% + 2.1% = 7.6%

股权成本 = 4.5% + 1.2 × 7.6% = **13.6%**

**步骤3：计算债务成本**

BB+评级违约利差：约2.5%

税前债务成本 = 4.5% + 2.5% = 7.0%
税后债务成本 = 7.0% × (1 - 34%) = **4.6%**

**步骤4：计算WACC**

$$
WACC = 60\% \times 13.6\% + 40\% \times 4.6\% = 8.2\% + 1.8\% = \textbf{10.0\%}
$$

::: {.callout-important}
## 小结：折现率的完整构成

$$
\text{股权成本} = R_f + \beta \times (ERP_{成熟市场} + CRP)
$$

$$
\text{债务成本} = R_f + \text{违约利差}
$$

$$
WACC = \frac{E}{V} \times R_e + \frac{D}{V} \times R_d \times (1-T)
$$

每个输入参数的来源：

- $R_f$：政府债券收益率（期限匹配）
- $ERP$：历史法或隐含法估算
- $CRP$：主权利差或评级查表
- 违约利差：公司评级或合成评级
:::

## 本章总结

本章讨论了估值中最基础也最关键的几个输入参数：

1. **无风险利率**是一切的起点，必须满足无违约风险和无再投资风险，实践中用期限匹配的政府债券利率

2. **股权风险溢价**可以用历史法或隐含法估算，隐含法更具前瞻性，当前美国市场的ERP约为5.5-6%

3. **国家风险溢价**补偿新兴市场的额外风险，可以通过主权利差、相对波动率或评级表来估算

4. **违约利差**决定债务成本，取决于公司信用评级和市场环境

5. **货币一致性**是容易犯错但绝对不能犯错的原则
