【第32章】价值衡量工具：EVA、CFROI与DCF的桥梁

金融
投资
估值
DCF模型虽然全面，但过于复杂且易被操纵。本章探讨两种替代的价值衡量机制——经济增加值(EVA)和现金流投资回报率(CFROI)，分析它们与DCF的数学等价性，以及管理者可能利用的三种’游戏’来提升指标却损害价值。

1 从一个问题开始

在第31章，我们建立了一个完整的 DCF 价值提升框架——通过提高现金流、增加增长、延长高增长期、降低资本成本来创造价值。这个框架在理论上是完美的，但在实践中面临一个根本性的挑战：

如何将这个复杂的、基于估计的框架，转化为一个简单的、可以用来考核管理层绩效的指标？

DCF 模型需要预测未来多年的现金流、估计永续增长率、计算资本成本——每一个输入都涉及判断和假设。如果管理层的薪酬与 DCF 估值挂钩，他们完全可以通过调整假设来”创造”任何想要的结果。

一个直觉上的替代方案是使用市场价格。如果市场是有效的，股价已经反映了所有关于未来现金流的信息，那么：

  • 股价上涨 → 管理层创造了价值 → 奖励
  • 股价下跌 → 管理层摧毁了价值 → 惩罚

这就是为什么股票期权和限制性股票成为管理层薪酬的标准组成部分。

但市场价格也有问题：

  1. 噪音：即使市场是有效的，股价也会围绕真实价值波动。有时候股价上涨，管理层获得奖励，但实际上价值正在被摧毁；反之亦然。
  2. 只适用于整体公司：市场价格无法用于评估公司内部的各个部门或业务单元。
  3. 时机敏感：股价受宏观经济、行业周期等外部因素影响，管理层可能因为运气好坏而被奖励或惩罚。

在这种背景下，经济增加值（EVA）和现金流投资回报率（CFROI）应运而生。它们试图在 DCF 的理论严谨性和市场价格的简单性之间找到平衡：

  • EVA：衡量现有投资创造的美元超额回报
  • CFROI：衡量现有投资获得的百分比回报

本章将深入探讨：

  • EVA 和 CFROI 如何计算？
  • 它们与 DCF 估值有什么数学关系？
  • 在什么条件下，使用这些指标可能导致增加指标却摧毁价值的决策？
  • 这些指标与市场价值有什么关系？

2 经济增加值（EVA）

2.1 为什么需要 EVA？

让我们从一个简单的问题开始：一家公司今年创造了多少价值？

传统会计给我们的答案是净利润。但净利润有一个根本性的缺陷：它忽略了股权资本的成本。

想象两家公司：

公司 A 公司 B
投入资本 $100M $100M
税后营业收入 $15M $15M
净利润（假设） $10M $10M
资本成本 10% 20%

从净利润角度，两家公司表现相同。但从价值创造角度：

  • 公司 A：资本回报率 15% > 资本成本 10% → 创造价值
  • 公司 B：资本回报率 15% < 资本成本 20% → 摧毁价值

EVA 正是为了捕捉这个差异而设计的。

2.2 EVA 的定义与计算

经济增加值（Economic Value Added, EVA）的定义是：

\[ \text{EVA} = (\text{资本回报率} - \text{资本成本}) \times \text{投入资本} \]

或者等价地：

\[ \text{EVA} = \text{税后营业收入} - \text{资本成本} \times \text{投入资本} \]

这两个公式是等价的，因为税后营业收入 = 资本回报率 × 投入资本。

直觉理解：EVA 就是”扣除资本成本后的利润”。它告诉你，在支付了所有资本提供者（包括股东）的”利息”后，公司还剩下多少。

2.3 EVA 的三个输入

计算 EVA 需要三个输入：

输入 1：投入资本（Capital Invested）

为什么不用市场价值？因为市场价值包含了未来增长的预期，而我们想衡量的是现有资产的表现。

因此，EVA 使用账面价值作为投入资本的代理。但账面价值需要调整：

调整项 原因
经营租赁资本化 经营租赁是隐藏的债务和资产
R&D 资本化 R&D 是投资，不是费用
一次性费用还原 避免管理层通过”大洗澡”操纵资本

公司越老，需要的调整就越多，因为需要追溯所有历史会计决策的影响。

输入 2：税后营业收入

同样需要对经营租赁、R&D 和一次性费用进行调整，以保持与投入资本的一致性。

输入 3：资本成本

这里出现一个看似矛盾的地方：

  • 投入资本使用账面价值
  • 资本成本使用市场价值权重

为什么？因为公司必须赚取超过市场价值资本成本的回报才能创造价值。使用账面价值资本成本会低估真实的资本成本，从而高估 EVA——而且对高杠杆公司的低估更严重。

注记EVA 的实践推广：Stern Stewart

1990年代，EVA 被纽约咨询公司 Stern Stewart 大力推广。公司创始人 Joel Stern 和 Bennett Stewart 成为这一指标的主要布道者。

在将 EVA 应用于实际公司的过程中，Stern Stewart 发现需要对会计指标进行修正才能得到更真实的超额回报估计。Bennett Stewart 在1991年的著作《The Quest for Value》中提到了一些调整，包括商誉调整、经营租赁转换等。

许多在这一时期采用 EVA 的公司也将管理层薪酬与 EVA 挂钩。因此，EVA 的定义和计量方式对各级管理者来说变得至关重要。

2.4 EVA、NPV 与 DCF 估值的等价性

这是本章最重要的理论结果之一：EVA 是 NPV 规则的自然延伸。

回顾传统公司金融中的 NPV 规则：

\[ \text{NPV} = -\text{初始投资} + \sum_{t=1}^{n} \frac{\text{现金流}_t}{(1+r)^t} \]

NPV > 0 的项目增加企业价值，NPV < 0 的项目减少企业价值。

关键定理：项目的 NPV 等于该项目各年 EVA 的现值之和：

\[ \text{NPV} = \sum_{t=1}^{n} \frac{\text{EVA}_t}{(1+k_c)^t} \]

这个等价性让我们可以将企业价值分解为三个部分：

\[ \boxed{ \text{企业价值} = \text{投入资本}_{\text{现有资产}} + \sum_{t=1}^{\infty} \frac{\text{EVA}_{t,\text{现有资产}}}{(1+k_c)^t} + \sum_{t=1}^{\infty} \frac{\text{EVA}_{t,\text{未来项目}}}{(1+k_c)^t} } \]

换言之：

\[ \text{企业价值} = \text{投入资本} + \text{PV(现有资产的EVA)} + \text{PV(未来投资的EVA)} \]

这个分解非常有洞察力：

  • 第一项：你已经投入的资本
  • 第二项：现有资产创造的超额价值（可能为正或负）
  • 第三项：未来投资机会创造的超额价值
注记Illustration 32.1：DCF 与 EVA 的等价性

考虑一家公司：

重要EVA 与 DCF 估值何时会不一致？

要让两种方法得出相同结果，必须满足以下条件：

  1. 一致的营业收入：用于 FCFF 和 EVA 的税后营业收入必须相同（同样的调整）

  2. 内生增长率：DCF 中的增长率必须从基本面估计： \[ g = \text{再投资率} \times \text{资本回报率} \] 如果增长率是外生输入且不满足这个关系，两种方法会给出不同结果。

  3. 一致的资本投入：EVA 中未来各期的投入资本必须通过”期初资本 + 当期再投资”来计算。

  4. 一致的终值假设：在 ROC = WACC 的特殊情况下，终值等于终值年初的投入资本。在更一般的情况下，需要确保终值年初的投入资本与永续 ROC 假设一致。

3 EVA 与企业价值：潜在冲突

现在我们来到本章最关键的部分：如果公司采用 EVA 作为绩效考核指标，管理者可能如何操纵这个指标来提升 EVA 却同时摧毁价值？

回顾企业价值的 EVA 分解：

\[ \text{企业价值} = \text{投入资本}_{\text{现有资产}} + \text{PV(现有资产EVA)} + \text{PV(未来投资EVA)} \]

管理者可以利用这个公式中的各个组成部分来玩”游戏”。

3.1 游戏一：投入资本游戏（The Capital Invested Game）

观察上面的公式：前两项（投入资本 + 现有资产 EVA 现值）都依赖于投入资本的计量。

关键洞察：如果投入资本减少，而营业收入不变，那么：

  • 第一项（投入资本）下降
  • 但第二项（EVA 现值）会成比例增加

让我们用 Illustration 32.1 中的公司来演示。假设投入资本被估计为 $50M 而不是 $100M，但营业收入仍为 $15M：

组成部分 原始（$100M资本） 调整后（$50M资本）
投入资本 $100.00M $50.00M
资本回报率 15% 30%
现有资产 EVA (.15−.10)(100)/.10 = $50M (.30−.10)(50)/.10 = $100M
未来投资 EVA $20.85M $20.85M
企业价值 $170.85M $170.85M

企业价值不变，但 EVA 从 $50M 增加到 $100M！

这告诉我们：当管理者被考核 EVA 时，他们有强烈的动机去减少投入资本的计量值。

有些减少投入资本的行动是真正创造价值的。例如，关闭一个不产生营业收入的工厂，释放的现金确实增加价值。

但有些行动是纯粹修饰性的：

  • 通过一次性重组费用减少资本
  • 用租赁代替购买（因为租赁的资本影响可能更小）

数值示例：假设 Illustration 32.1 中的管理者用租赁资产替换了一半的自有资产。租赁资产的资本计量只有 $40M（低于被替换资产的 $50M），但年营业收入从 $15M 降到 $14.8M：

组成部分 价值
投入资本 $90.00M
现有资产 EVA (.1644−.10)(90)/.10 = $58.00M
未来投资 EVA $20.85M
企业价值 $168.85M

企业价值从 $170.85M 下降到 $168.85M（减少 $2M），但 EVA 从 $50M 增加到 $58M（增加 $8M）！

警告部门资本分配的政治

当 EVA 在部门层面计算时，各部门的投入资本是根据预设规则（如收入、员工数量）分配的。

虽然我们希望这些规则客观公正，但它们往往是主观的，会过度分配资本给某些部门而对其他部门分配不足。

由于部门之间存在对边际投资的竞争，这些分配很可能反映各部门影响决策过程的政治能力。因此，被分配较少资本的部门会高估 EVA，被分配较多资本的部门会低估 EVA。

3.2 游戏二：未来增长游戏（The Future Growth Game）

企业价值是现有资产价值 + 未来增长价值。但当管理者被考核当年 EVA 或年度 EVA 变化时，被衡量的只是现有资产的 EVA。

因此，管理者可能会牺牲未来增长来换取更高的当前 EVA。

数值示例：回到 Illustration 32.1 的公司。假设管理者可以采取行动将现有资产的资本回报率从 15% 提高到 16%，但代价是未来投资的回报率从 15% 降到 12%：

组成部分 原始 增长权衡后
投入资本 $100.00M $100.00M
现有资产 EVA (.15−.10)(100)/.10 = $50M (.16−.10)(100)/.10 = $60M
年1投资 EVA [(.15−.10)(10)/.10] = $5M [(.12−.10)(10)/.10] = $2M
年2投资 EVA $4.55M $1.82M
年3投资 EVA $4.13M $1.65M
年4投资 EVA $3.76M $1.50M
年5投资 EVA $3.42M $1.37M
企业价值 $170.85M $168.34M

企业价值从 $170.85M 下降到 $168.34M，但当前 EVA 增加了 $10M！

更阴险的是，让我们看看未来几年的年度 EVA：

         原始EVA    增长权衡后EVA
Year 1:  $5.00M     $7.00M ← 更高
Year 2:  $5.50M     $6.20M ← 更高
Year 3:  $6.05M     $5.45M ← 仍然增长
Year 4:  $6.66M     $4.75M
Year 5:  $7.32M     $4.10M

增长权衡虽然导致更低的企业价值，但前三年的 EVA 都更高！

一些基于 EVA 的薪酬机制试图惩罚牺牲未来增长的管理者：管理者的部分薪酬被放入”薪酬银行”，只有在几年后才能提取。但这种方法有严重局限：

  1. 管理者任期有限：这种机制最多只能考察未来3-4年的 EVA
  2. 只能惩罚明显的权衡：在更微妙的情况下——EVA 继续增长但增速低于本应达到的水平——很难设计惩罚机制

3.3 游戏三：风险转移游戏（The Risk-Shifting Game）

企业价值 = 投入资本 + EVA 现值。后者不仅取决于 EVA 的美元金额，还取决于资本成本。

一家公司可以通过投资风险更高的项目来增加 EVA，但如果这些投资增加了经营风险和资本成本，企业价值可能反而下降。

数值示例：假设 Illustration 32.1 的公司能够将现有资产和未来投资的回报率都从 15% 提高到 16.25%，第5年后从 10% 提高到 11%。但同时，资本成本从 10% 上升到 11%：

组成部分 原始（WACC=10%） 高风险策略（WACC=11%）
投入资本 $100.00M $100.00M
现有资产 EVA (.15−.10)(100)/.10 = $50M (.1625−.11)(100)/.11 = $47.73M
年1投资 EVA $5.00M $4.77M
年2投资 EVA $4.55M $4.30M
年3投资 EVA $4.13M $3.87M
年4投资 EVA $3.76M $3.49M
年5投资 EVA $3.42M $3.14M
企业价值 $170.85M $167.31M

但让我们看看年度 EVA：

年份 原始EVA 高风险EVA
1 $5.00M $5.25M ← 更高
2 $5.50M $5.78M ← 更高
3 $6.05M $6.35M ← 更高
4 $6.66M $6.99M ← 更高
5 $7.32M $7.69M ← 更高

每一年的 EVA 都更高，但企业价值却下降了！

风险效应（更高的折现率）超过了更高的美元超额回报，导致价值下降。

这种风险转移对于仅基于年度 EVA 变化来考核的公司尤其危险。如果计量的资本成本没有反映风险变化或滞后于风险变化，管理者会倾向于转向风险更高的投资。

重要EVA 对高增长公司的局限

EVA 是一种偏向现有资产、远离未来增长的方法。当 EVA 在部门层面计算时，高增长部门往往有最低的 EVA，有时甚至是负 EVA。

对于 80-90% 的价值来自未来增长潜力的高增长科技公司，使用 EVA 有三个问题：

  1. 资本计量问题被放大：科技公司的会计资本投入问题更严重
  2. 增长权衡风险更大：当大部分价值来自未来增长时，牺牲增长换取当前 EVA 的诱惑更大
  3. 风险转移更容易：这些公司经历的持续变化使它们更容易进行风险转移

3.4 EVA 与市场价值

EVA 增加会导致市场价值增加吗？

一般来说，EVA 增加会导致企业价值增加（排除前面讨论的增长和风险游戏）。但它不一定会导致股价上涨。

原因是：市场价格已经内嵌了对未来 EVA 的预期。

  • 对于 Apple 这样的公司，市场定价基于它将持续赚取巨大且增长的 EVA 的假设
  • 企业市值是否因 EVA 公告而上涨或下跌，在很大程度上取决于预期的 EVA 变化是多少
公司类型 预期 EVA实际变化 市值反应
成熟公司 无变化或下降 增加 上涨
高增长公司 大幅增加 增加但低于预期 下跌
高增长公司 大幅增加 增加且超预期 上涨

这对投资者来说不应该陌生——几十年来，盈利公告都是相对于预期来判断的，“盈利惊喜”才是驱动价格的因素。

实证证据：

Merrill Lynch 的 Richard Bernstein 研究了 EVA 与股票收益的关系：

  • 1987-1997年间，EVA 绝对水平最高的50家公司，年化收益率 12.9%，而同期 S&P 500 年化收益率 13.1%
  • EVA 增长率最高的50家公司，年化收益率 12.8%，同样跑输 S&P 500

结论：EVA 最高或 EVA 增长最快的股票，不一定能为股东带来高回报。

3.5 股权 EVA（Equity EVA）

传统 EVA 使用总资本，但可以修改为股权指标：

\[ \text{股权 EVA} = (\text{股权回报率} - \text{股权成本}) \times \text{股权投入} \]

\[ = \text{净利润} - \text{股权成本} \times \text{股权投入} \]

为什么使用股权 EVA？

回顾第21章对金融服务公司的讨论：定义债务（因此也是资本）可能面临计量问题，因为公司的大部分都可能被归类为债务。因此，金融服务公司应该使用股权估值模型。

将这一论点延伸到 EVA：股权 EVA 是金融服务公司比传统 EVA 更好的绩效衡量指标。

但需要注意：传统 EVA 的所有问题（投入资本游戏、增长游戏、风险游戏）同样影响股权 EVA。银行和保险公司可以像其他公司玩传统 EVA 游戏一样玩股权 EVA 游戏。

4 现金流投资回报率（CFROI）

4.1 为什么需要 CFROI？

EVA 衡量的是美元超额回报。但美元金额受公司规模影响——一家 $100B 的公司创造 $1B 的 EVA 和一家 $10B 的公司创造 $1B 的 EVA，含义完全不同。

现金流投资回报率（Cash Flow Return on Investment, CFROI）衡量的是百分比回报——现有投资的内部收益率（IRR），基于实际（而非名义）现金流。

4.2 CFROI 的计算

CFROI 使用四个输入：

输入 定义 说明
总投资（GI） 现有资产的总投资额 账面价值 + 累计折旧 + 通胀调整
总现金流（GCF） 当年资产产生的总现金流 税后营业收入 + 折旧摊销
资产寿命（n） 原始投资时资产的预期寿命 因行业而异
残值（SV） 资产寿命末期的预期残值（当前美元） 通常是非折旧部分（土地、建筑）

CFROI 是使这些现金流的净现值等于总投资的内部收益率：

CFROI = IRR that makes PV of GCF & SV = GI

       GCF    GCF    GCF    GCF           GCF+SV
        |      |      |      |              |
GI     1      2      3      4    ...       n

替代公式：使用经济折旧

有一种替代方法是设置一个年金来覆盖资产的预期重置成本：

\[ \text{经济折旧} = \frac{\text{当前美元重置成本} \times k_c}{(1+k_c)^n - 1} \]

其中重置成本 = 总投资 - 残值。然后：

\[ \text{CFROI} = \frac{\text{总现金流} - \text{经济折旧}}{\text{总投资}} \]

数值示例：

假设一项资产： - 总投资：$2,431M - 总现金流：$390M/年 - 残值（当前美元）：$607.8M - 资产寿命：10年

传统 CFROI 方法：

         $390   $390   $390   $390              $390+$607.8
          |      |      |      |                    |
-$2,431   1      2      3      4    ...            10

IRR = 11.71%

经济折旧方法：

\[ \text{经济折旧} = \frac{(2,431 - 607.8) \times 0.08}{1.08^{10} - 1} = \$125.86M \]

\[ \text{CFROI} = \frac{390.00 - 125.86}{2,431.00} = 10.87\% \]

两种方法的差异源于再投资率假设：

  • 传统方法：中间现金流以 IRR 再投资
  • 经济折旧方法：设置用于重置的现金流以资本成本再投资

如果使用 IRR = 11.71% 计算经济折旧，两种方法会得到相同结果。

4.3 CFROI、IRR 与 DCF 估值的关系

如果 NPV 是 EVA 方法的基础，那么 IRR 是 CFROI 方法的基础。

项目的 IRR：

         ATCF   ATCF   ATCF   ATCF          ATCF+SV
          |      |      |      |              |
Initial   1      2      3      4    ...      n
Investment

IRR 是使现金流现值等于初始投资的折现率。

乍一看，CFROI 做的是同样的事情——它使用总投资作为初始投资的等价物，假设总现金流在项目寿命内保持不变，并计算实际 IRR。

但有两个重要差异：

差异1：现金流模式

  • IRR 不要求现金流恒定（即使是实际值）
  • CFROI 假设实际现金流不随时间变化

这对成熟行业的投资可能是合理假设，但如果存在实际增长，会低估项目回报。

差异2：视角

  • IRR 基于增量未来现金流，已发生的现金流被视为”沉没”
  • CFROI 试图重构项目，使用已发生和未来的现金流

数值示例：

考虑前面描述的项目。在原始投资时，IRR 和 CFROI 都是 11.71%。

现在假设我们在项目进行3年后计算。CFROI 仍然是 11.71%（因为原始输入没变）。

但如果资产的市场价值已经上涨到 $2.5B，IRR 会变为：

         $390   $390   $390   $390          $390+$607.8
          |      |      |      |                |
-$2,500   1      2      3      4    ...        7

IRR = 6.80%

现在出现了矛盾：

  • CFROI（11.71%）> 资本成本（8%）→ 公司运用资产良好
  • IRR（6.80%）< 资本成本（8%）→ 股东最好卖掉资产

哪个是对的？

IRR 是对的！如果 IRR 低于资本成本，公司所有者卖掉资产获取市场价值，会比继续运营更好。CFROI 超过资本成本不意味着公司运用资产良好——它只是反映了历史的、而非当前的投资机会。

4.4 将 CFROI 与企业价值联系起来

对于稳定增长公司：

\[ \text{企业价值} = \frac{\text{预期FCFF}}{k_c - g} \]

可以近似改写为 CFROI 形式：

\[ \text{企业价值} = \frac{(\text{CFROI} \times \text{GI} - \text{DA})(1-t) - (\text{CX} - \text{DA}) - \Delta\text{WC}}{k_c - g} \]

数值示例：

一家公司：CFROI = 30%，总投资 = $100M，资本支出 = $15M，折旧 = $10M，无营运资本需求，资本成本 = 10%，税率 = 40%，稳定增长率 = 5%：

\[ \text{企业价值} = \frac{(0.30 \times 100 - 10)(1-0.4) - (15-10) - 0}{0.10 - 0.05} = \$140M \]

关键洞察：企业价值不仅取决于 CFROI，还取决于总投资、税率、增长率、资本成本和再投资需求。

4.5 CFROI 的创新：Fade Factor

CFROI 实践者最大的贡献是对资本回报率如何随时间向资本成本回归的研究。

Holt Associates（CFROI 的主要支持者之一）允许使用 fade factor：当前 CFROI 随时间向实际资本成本回归。fade factor 是通过观察不同 CFROI 类别的公司并追踪其随时间变化来实证估计的。

例如，一家当前 CFROI = 20%、实际资本成本 = 8% 的公司，预计其 CFROI 会随时间下降。

在这种更复杂的框架下，企业价值可以写为：

  1. 现有资产剩余寿命内的现金流现值： \[ \sum_{t=1}^{n} \frac{\text{CFROI}_{\text{aip}} \times \text{GI}_{\text{aip}}}{(1+k_c)^t} \]

  2. 未来投资超额现金流的现值： \[ \sum_{t=1}^{\infty} \frac{\text{CFROI}_{t,\text{NI}} \times \Delta\text{GI}_t}{(1+k_c)^t} - \Delta\text{GI}_t \]

如果 CFROI_{t,NI} = k_c，第二部分等于零。

因此，企业价值取决于：

  • 现有资产的 CFROI
  • CFROI 向资本成本回归的速度和突然程度

公司可以通过以下方式增加价值：

  • 在给定总投资下增加现有资产的 CFROI
  • 减缓 CFROI 向资本成本回归的速度

这与我们在第31章的分析完全一致：现有投资的现金流（提高当前 CFROI）、高增长期长度（减缓 fade 速度）、增长期内的增长率（保持超额回报不急剧下降）。

4.6 CFROI 与企业价值：潜在冲突

与 EVA 类似，管理者可以采取增加 CFROI 同时减少企业价值的行动：

1. 减少总投资

如果减少现有资产的总投资，CFROI 可能增加。但因为决定价值的是 CFROI 乘以总投资，公司可能增加 CFROI 却降低价值。

2. 牺牲未来增长

CFROI 比 EVA 更聚焦于现有资产，不考虑未来增长。管理者可能以牺牲未来增长为代价来增加 CFROI，导致价值下降而 CFROI 上升。

3. 风险权衡

虽然 CFROI 与实际资本成本比较来判断是否创造或摧毁价值，但这只是对风险的部分修正。企业价值仍然是预期未来现金流的现值。公司可能增加 CFROI 与资本成本的差距，但如果更高资本成本的现值效应超过了更高 CFROI，仍然会损失价值。

总之，CFROI 增加本身并不表明企业价值增加——它可能以牺牲增长和/或增加风险为代价。

4.7 CFROI 与市场价值

CFROI 与市场价值之间存在关系——高 CFROI 的公司通常有高市值。这并不令人惊讶，因为它反映了我们之前关于 EVA 的观察。

但创造回报的是市值变化，而非市值本身。当涉及市值变化时，CFROI 与价值变化的关系往往更弱。

由于市值反映预期，没有理由相信高 CFROI 的公司会赚取超额回报。CFROI 变化与超额回报的关系更有趣：如果 CFROI 增加被视为正面惊喜，CFROI 增幅最大的公司应该赚取超额回报。但实际上，CFROI 的实际变化必须与预期相比较。

5 价值提升的反思

企业价值有三个组成部分：

  1. 现有资产产生现金流的能力
  2. 再投资创造未来增长的意愿和质量
  3. 资本成本

要创造价值，公司必须：

  • 从现有资产产生更高现金流，不影响增长前景或风险状况
  • 更多再投资且获得更高超额回报，不增加资产风险
  • 降低为现有资产或未来增长融资的成本，不降低投资回报

所有价值提升措施都是这些简单主题的变体。

无论是衡量美元超额回报（如 EVA）还是百分比超额回报（如 CFROI），这些方法之所以受欢迎，是因为它们看起来比 DCF 估值更简单、更少主观。但这种简单性是有代价的——这些方法对价值的其他组成部分做了微妙的假设，而这些假设往往不被许多用户注意或认识到。

强调 EVA 并奖励管理者增加 EVA 的方法，往往假设 EVA 的增加不是以牺牲未来增长或增加风险为代价的。基于 CFROI 判断绩效的实践者也做出类似假设。

★ Insight ─────────────────────────────────────

新价值衡量工具的真正价值

EVA 和 CFROI 在更大的估值背景中是有价值的：

  1. 更好的 ROC 估计：EVA 建议的对营业收入的调整，加上 CFROI 的现金流回报视角，可以帮助我们得到更好的资本回报率估计——这是传统估值模型中预期增长的关键输入

  2. 更易处理的终值：终值计算在传统模型中往往是假设小变化导致价值大变化的敏感点。如果我们从超额回报角度思考，而不仅仅是增长率和折现率，终值会更容易处理

  3. 实证 Fade Factor：CFROI 实践者收集的关于 fade factor 的实证证据，对传统估值模型非常有价值——传统实践者有时错误地假设当前回报会永远持续

─────────────────────────────────────────────────

5.1 价值提升的现实

在审视各种价值提升方法时，我们应该考虑几个事实：

第一，除非管理者真正承诺以价值最大化为首要目标，否则没有任何价值提升机制会有效。如果管理者把其他目标放在首位，任何机制都不会起作用。反之，如果管理者真正关心价值最大化，他们几乎可以让任何机制为他们服务。

第二，虽然将我们选择的价值提升指标与管理层薪酬挂钩是合理的，但这有一个缺点：管理者会随时间聚焦于让自己在该指标上看起来更好，即使这只能通过减少企业价值来实现。

第三，没有创造价值的魔法子弹。在竞争市场中，价值创造是艰苦的工作，几乎总是涉及成本和收益的权衡。每个人都在价值创造中发挥作用，这绝不仅仅是财务分析师的专属领域。事实上，金融工程创造的价值比好的战略、营销、生产或人事决策创造的价值要小得多、也不那么重要。

6 总结

重要核心要点

  • EVA 与 NPV 等价：项目的 NPV = 各年 EVA 现值之和；企业价值 = 投入资本 + 现有资产 EVA 现值 + 未来投资 EVA 现值

  • EVA 的三种操纵游戏：

    • 投入资本游戏：减少资本计量，提高 EVA 但不改变价值
    • 未来增长游戏：牺牲未来增长换取当前 EVA
    • 风险转移游戏：增加风险提高 EVA 但损害价值

  • CFROI 是现有投资的实际 IRR：基于总投资、总现金流、资产寿命和残值计算；应与实际资本成本比较

  • CFROI vs IRR：CFROI 基于历史投资，IRR 基于当前市场价值；当两者冲突时，IRR 更能反映当前的投资机会

  • EVA/CFROI 与市场价值：高 EVA/CFROI 公司不一定有高股票回报；市场关注的是相对预期的变化

  • Fade Factor：超额回报会随时间向资本成本回归；这是 CFROI 实践者的重要贡献

指标 衡量内容 基础理论 主要风险
EVA 美元超额回报 NPV 投入资本操纵、牺牲增长、风险转移
CFROI 百分比回报（实际） IRR 减少总投资、牺牲增长、风险权衡

本章我们回答了开头的问题：如何将复杂的 DCF 框架转化为简单的绩效考核指标？

答案是：EVA 和 CFROI 提供了这种简化，但简化是有代价的。它们对价值的其他组成部分做了微妙的假设，而这些假设往往不被认识到。管理者可以利用这些假设来提升指标却摧毁价值。

最令人清醒的现实：价值创造是艰苦的工作，涉及权衡，没有魔法子弹。真正的价值不是来自财务工程，而是来自好的战略、运营、营销和人事决策。

7 思考题

  1. EVA 与净利润：解释为什么净利润不是衡量价值创造的好指标。EVA 如何改进这一点？EVA 自身又有什么局限？

  2. 数学等价性：用一个简单的两年项目，验证 NPV = EVA 现值之和的等价性。如果增长率是外生输入而不是从再投资率和 ROC 推导，这个等价性会打破吗？

  3. 投入资本游戏：假设一家公司通过售后回租减少了账面资本 $50M，但年租金增加 $6M（税前）。如果资本成本是 10%，税率是 40%，这个交易是创造价值还是摧毁价值？EVA 会如何变化？

  4. 高增长公司的困境：为什么 EVA 对高增长科技公司可能是一个特别差的绩效指标？如果你是 Amazon 的董事会成员，会如何修改 EVA 框架来更好地激励管理层？

  5. CFROI vs IRR：一项投资3年前以 $100M 购买，当前 CFROI 是 15%，实际资本成本是 10%。资产当前市场价值是 $130M，剩余寿命5年，每年实际现金流 $15M，残值 $30M。计算 IRR 并解释与 CFROI 的差异意味着什么。

  6. 跨章节联系：本章讨论的 EVA 三种”游戏”与第31章的价值创造四条路径有什么关系？一个好的绩效考核体系应该如何设计来防止这些游戏？
